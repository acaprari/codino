Story-Driven Path with Element Choices

How branching and element selection could work together

Option A: Branches ARE Element Choices

When the path branches, each branch represents choosing a different element. Simpler - one choice per level.

Example: After completing level 2
How it works: Player clicks one of the three branches. That choice determines which element is added to the story AND which path the journey continues on.

Option B: Node Selection + Element Popup

Branches lead to next level, but when player clicks the node, they choose from 2-4 elements via a popup/modal. More UI steps.

Example: Player reaches level 3 node
